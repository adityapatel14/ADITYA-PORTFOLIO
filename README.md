# 💼 Data Analyst Portfolio Website

A modern, interactive portfolio website showcasing my work, skills, and projects as a **Junior Data Analyst**.
Built with a focus on clean UI, real-world projects, and an engaging developer-style terminal interface.

---

## 🚀 Live Features

* 📊 Project showcase with real data analysis work
* 💻 Interactive terminal (CLI-style interface)
* 📄 Resume download integration
* 🔗 Quick access to GitHub, LinkedIn, and contact
* 📈 Realistic stats and metrics (no placeholders)
* 🎯 Optimized for recruiters and hiring managers

## 💻 Interactive Terminal

A unique feature of this portfolio is a built-in **command-line interface**.

### Available Commands:

```
help        → List all commands  
about       → About me  
skills      → View technical skills  
projects    → View all projects  
github      → GitHub profile  
linkedin    → LinkedIn profile  
resume      → View/download resume  
hire        → Contact information  
clear       → Clear terminal  
echo <text> → Repeat input  
```
